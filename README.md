Feedstock for [nnpdf](https://nnpdf.mi.infn.it/) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

NNPDF analysis framework

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `4.0.9` |
| linux-arm64 (`linux-aarch64`) | ✅ `4.0.9` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge nnpdf
```
